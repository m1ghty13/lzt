USAGE
=================================================================

Single account:
  python 1.py 1

Batch (accounts 1-10):
  for /L %i in (1,1,10) do python 1.py %i

With CAPTCHA API key:
  set CAPTCHA_API_KEY=your_key
  python 1.py 1

=================================================================
SUCCESS CRITERIA
=================================================================

A successful run is defined as:
- Script reaches the recovery form
- All form fields are filled
- Submit button is clicked
- No errors or timeouts
- Browser closes cleanly
- Log file shows completion

Exit code 0 = Success
Exit code 1 = Failure
