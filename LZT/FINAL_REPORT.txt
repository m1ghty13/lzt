HOW TO USE
================================================================

1. Single account:
   python 1.py 1

2. When CAPTCHA appears:
   - Check browser window
   - Click the checkbox "Verify you are human"
   - Wait for Cloudflare to complete
   - Script waits up to 3 minutes
   - Script continues automatically

3. Script completes when:
   - Form is filled and submitted
   - Contact button is clicked (if present)
   - Script exits with status "RECOVERY COMPLETED SUCCESSFULLY!"

4. Check logs:
   results/log_1.txt - detailed execution log
   results/log_2.txt - for second account, etc.

================================================================
SUPPORT / TROUBLESHOOTING
================================================================

Error: "Session is invalid"
→ Solution: Took more than 20 minutes. Increase SPEED_FACTOR or reduce delays.

Error: "CAPTCHA timeout"
→ Solution: You have 3 minutes to solve. Solve faster or increase timeout in code.

Error: "Form not found"
→ Solution: Wait for "Page loaded!" message. Page loads from Cloudflare take time.

Error: "503 Service Unavailable"
→ Solution: Restart Kameleo Desktop and check proxy settings.

Check logs: results/log_N.txt always has detailed info about what went wrong!
